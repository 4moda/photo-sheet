# Product Decisions (Current)

## ツールバー / パネル

- フローティングバーは `見た目 / 調整 / タイトル / 書き出し`（上限 6 個。2026-07-17 に「調整」を追加）
- パネル外タップで閉じる
- 写真操作はタップメニュー + 長押しドラッグ並べ替え
- シート全体の操作（撮影順に並べ替え・すべて削除）は右上 [+] メニューに集約

## デフォルト値

- 列数: `6`
- 用紙: `8x10`（`print8x10`）
- 背景:
  - `grid`: 白
  - `filmStrip`: 黒
- 仕上げ調整: すべてニュートラル（OFF）
- 撮影日キャプション自動表示: OFF（フィルム風のシンプルな見た目を既定で維持する。オーナー判断 2026-07-17）

## 役割分離

- 見た目パネル: 余白・間隔・背景など、レイアウトの設定
- 調整パネル: シート全体の仕上げ（モノクロ / コントラスト / 粒状感 / フェード / 色温度 / 周辺減光）。
  写真ごとのレタッチは提供しない。粒状感・周辺減光はネガ / レンズ由来なので写真単位に適用される
- タイトルパネル: タイトル・サブタイトル・撮影日自動表示トグル
- 書き出しパネル（画像）: 用紙フォーマット
- 書き出しパネル（動画）: 速度・表示行数・全体表示

## 取り込みと並び順（2026-07-17）

- 取り込み時は撮影順に自動整列: EXIF 撮影日時 → なければファイル名の自然順
  （フィルムスキャンは EXIF が無く、ファイル名 = スキャン順 = 撮影順のことが多いため）
- 手動並べ替え済みのシートは崩さない（追加分だけ末尾に撮影順で追記）
- 全体の並べ替えは [+] メニューの「撮影順に並べ替え」で明示的に実行

## スタイル

- `グリッド`（インデックスプリント風）/ `フィルム`（ベタ焼き風）/ `スリーブ`（ネガファイルのポケット風。2026-07-17 追加）
- スリーブの既定背景は `paperGray`（アーカイブ台紙のイメージ）。フィルム選択（35mm 等）はフィルムスタイルと共有

## フィルムフォーマット

- 35mm フルフレーム / ハーフ / 120（6×6・6×7）
- 120 はパーフォレーション（スプロケット穴）なし・コマ番号は「8A」併記なしで描く（実物準拠）

## フィルム的ディテール（すべて default OFF。シンプルな見た目を既定で維持）

- デート焼き込み: クォーツデート風のオレンジ日付（`'26 7 17`）を各コマ右下に刻印。EXIF 撮影日がある写真のみ
- 縁のコマ番号刻印: エッジテキストに `▸12` を併記
- エッジテキストのプリセット: 実在フィルム銘柄は商標リスクがあるため汎用表記のみ（COLOR 400 / MONO 400 等）

## 動画 UX

- 進捗オーバーレイ表示（%）
- 書き出し処理は CVPixelBuffer 直接描画で高速化済み

## やらないこと（復活させない）

- **赤いマークで写真をセレクトした風に見せる機能**（オーナー判断 2026-07-17。
  参考にした Web サービスの模倣になりすぎるため採用しない）
- 写真ごとの個別レタッチ（調整はシート全体の仕上げに限定する）
