# Product Decisions (Current)

## ツールバー / パネル

- フローティングバーは `見た目 / タイトル / 書き出し`
- パネル外タップで閉じる
- 写真操作はタップメニュー + 長押しドラッグ並べ替え

## デフォルト値

- 列数: `6`
- 用紙: `8x10`（`print8x10`）
- 背景:
  - `grid`: 白
  - `filmStrip`: 黒

## 役割分離

- 見た目パネル: 余白・間隔・背景など、表示/動画にも効く設定
- 書き出しパネル（画像）: 用紙フォーマット
- 書き出しパネル（動画）: 速度・表示行数・全体表示

## 動画 UX

- 進捗オーバーレイ表示（%）
- 書き出し処理は CVPixelBuffer 直接描画で高速化済み
